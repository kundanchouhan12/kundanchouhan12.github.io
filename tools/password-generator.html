What is the Password Generator?

The Password Generator is a free, browser-based security tool that creates strong, random passwords using cryptographically secure randomness provided by the Web Cryptography API (window.crypto.getRandomValues). Unlike pseudo-random number generators, this method produces entropy that is suitable for security-sensitive applications such as account passwords, encryption keys, and API secrets. You can customise every aspect of the generated password — length, character sets, exclusion of visually similar characters, and whether characters can repeat.

Cybersecurity experts universally recommend using a different strong password for every account. Data breaches expose billions of credentials annually, and attackers use credential stuffing attacks to test stolen username/password pairs across hundreds of websites. A randomly generated 16-character password with mixed case, numbers, and symbols takes effectively infinite time to crack by brute force, providing robust protection for your accounts.

How to Use — Step-by-Step

  1. Set your desired password length using the slider (4–128 characters).
  2. Check or uncheck character type options: Uppercase (A–Z), Lowercase (a–z), Numbers (0–9), and Symbols.
  3. Optionally enable Exclude Similar to remove confusable characters like I, l, 1, O, and 0.
  4. Enable No Repeating Chars if you want every character to appear only once.
  5. Set the Quantity field to generate multiple passwords at once (up to 50).
  6. Click Generate Password and use the copy button to grab your new password.

Frequently Asked Questions

What makes a strong password?
A strong password has: 12+ characters, mixed uppercase and lowercase, numbers, and symbols. No dictionary words, names, or predictable patterns. Never reused across accounts. A 16-character random password has over 7 quadrillion combinations — brute-force attacks are practically impossible.
How long should a password be?
Minimum by account type: general accounts 12 chars, email/social media 16 chars, banking/work 20+ chars, password manager master password 24+ chars. NIST recommends at least 8 characters but advises 15+ for better security. Longer is always stronger.
What is the difference between a password and a passphrase?
A password is a short random string (Xk#9mP!2). A passphrase is 4+ random words (correct-horse-battery-staple). Passphrases are 30+ characters and easier to type but equally secure. Use passwords (stored in a manager) for most accounts; use passphrases for your master password or anything you must type manually.
Should I use a password manager?
Yes — security experts universally recommend it. A password manager (Bitwarden, 1Password) generates and stores unique passwords for every account. Password reuse is the #1 account compromise risk — if one site is breached, attackers test the same password everywhere. A manager eliminates this entirely.
How often should I change my password?
NIST's current guidance no longer recommends routine periodic changes. Change your password when: you suspect it's compromised, a service you use reports a breach, or you've shared it with someone who no longer needs access. Forced regular changes often produce weaker passwords (Password1! → Password2!).
What is password entropy?
Entropy measures unpredictability in bits. Formula: bits = log₂(pool size) × length. A 16-char password from 94 characters ≈ 105 bits of entropy. Security experts recommend 72+ bits minimum. Each extra character adds ~6–7 bits. Higher entropy = exponentially harder to crack.
Is it safe to use an online password generator?
Yes, if it uses cryptographic randomness and never transmits passwords. This tool uses window.crypto.getRandomValues() — the Web Cryptography API standard. Passwords are never sent to any server, logged, or stored. They exist only in your browser and disappear when you close the tab.
Are generated passwords stored anywhere?
No. Passwords are never transmitted, stored, or logged anywhere. All generation happens in your browser session using crypto.getRandomValues(). They disappear permanently when you close the tab or refresh the page.
